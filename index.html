关于 Chrome Extension Downloader
由于种种原因,Chrome浏览器的扩展无法正常在线安装,或者需要将扩展保存以备下次安装使用. Chrome Extension Downloader 可以帮助你将任何Chrome浏览器扩展保存成crx安装包.
如何使用 Chrome Extension Downloader
  1. 打开Chrome网上应用商店
  2. 点击任意扩展程序,复制Chrome浏览器地址栏中扩展程序对应的地址. 类似于:
    https://chrome.google.com/webstore/detail/checker-plus-for-gmail/oeopbcgkkoapgobdbedcemjljbihmemj
    ( 地址中的 https 不能省略. )
  3. 将复制好的地址粘贴于上方的输入框内,点击下载按钮即可
如何安装crx文件
  1. 点击Chrome浏览器右上角的菜单按钮.依次点击 "更多工具"---"扩展程序",打开扩展程序页面.( 由于Chrome浏览器的版本不同,菜单的位置和名称可能略有不同,比较通用的做法是直接使用Chrome内置地址来打开扩展程序页面.做法是在Chrome地址栏直接输入 chrome://extensions/ 进行访问即可. )
  2. 在打开的扩展程序页面,将之前保存的扩展安装包crx文件拖放到此页面.然后根据提示进行安装即可. ( 值得注意的是,某些Chrome扩展程序强制要求必须从应用商店安装.则此种安装方式不可用. )
